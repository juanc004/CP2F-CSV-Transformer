
# Contributing to CSV Column Transformer

## Code of Conduct
This project and everyone participating in it is governed by the [CSV Column Transformer Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## How to Contribute
1. Fork the project repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them to your branch.
4. Create a pull request and describe your changes.

## Pull Request Process
1. Ensure that your code has been linted and passes all tests.
2. Update the README.md with details of changes to the interface, if applicable.
3. Your pull request will be reviewed by the maintainers, who may request some changes. Work on the changes and update your pull request accordingly.
4. Once approved, the maintainers will merge your pull request.
